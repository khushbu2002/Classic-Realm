# Classic Realm 🎮
** Multiplayer Android Gamming Application**

Classic Realm is a real-time multiplayer Android game built using Java and Firebase.  
The project focuses on backend logic for multiplayer interactions, real-time data synchronization, and user session management.

---

## 🛠️ Tech Stack

- **Language:** Java
- **Backend / Realtime Database:** Firebase
- **Platform:** Android

---

## ✨ Features

- Real-time multiplayer gameplay
- User authentication and session handling
- Real-time data synchronization using Firebase
- Backend logic for game state updates
- Event-driven gameplay interactions

## 📂 Project Structure (High Level)

- `activities/` – Game and UI activities
- `models/` – Game and user data models
- `services/` – Backend logic and Firebase handlers
- `utils/` – Helper and utility classes
